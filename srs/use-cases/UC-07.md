# UC-07: Add to Cart

## Use Case ID
UC-07

## Use Case Name
Add to Cart

## Primary Actors
Buyer

## Preconditions
- Buyer is logged in.
- Product is in stock.

## Postconditions – Success
- Product added to shopping cart.

## Postconditions – Failure
- Product not added.

## Main Flow
1. Buyer selects quantity.
2. Buyer clicks **Add to Cart**.
3. System validates stock.
4. System updates cart.
5. System displays confirmation.

## Alternative Flows
### AF-01
If stock unavailable, system shows error.

## Business Rules
- Cart persists across sessions.
