# UC-11: Manage Seller Dashboard

## Use Case ID
UC-11

## Use Case Name
Manage Seller Dashboard

## Primary Actors
Seller

## Preconditions
- Seller is logged in.

## Postconditions – Success
- Seller manages listings and orders.

## Main Flow
1. Seller opens dashboard.
2. System displays sales overview.
3. Seller manages listings.
4. Seller updates stock.
5. Seller manages orders.
6. Seller views analytics.

## Alternative Flows
### AF-01
No sales data available.

## Business Rules
- Only seller-owned listings can be managed.
